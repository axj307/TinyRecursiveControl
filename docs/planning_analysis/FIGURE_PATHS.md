## How to Update

When you re-run experiments and generate new planning analysis figures:

1. **Update the Experiment ID** and **Base Path** above
2. **Verify figure filenames** haven't changed (they shouldn't)
3. All guide documents (`DOUBLE_INTEGRATOR_GUIDE.md`, `VAN_DER_POL_GUIDE.md`, etc.) will reference the updated paths

## Figure Generation Script

Planning analysis figures are generated by:
```bash
python scripts/visualize_planning.py \
    --checkpoint outputs/experiments/{problem}_{method}_*/training/best_model.pt \
    --test_data data/{problem}/{problem}_dataset_test.npz \
    --problem {problem_name} \
    --output_dir outputs/experiments/{problem}_{method}_*/planning_analysis \
    --num_samples 100 \
    --level all
```

**Script location**: `scripts/visualize_planning.py:1`
